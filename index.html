Latest Project

Weather Information Centre

The Weather Information Center is a web application that provides real-time weather updates and forecasts for various locations. Developed using HTML, CSS, and JavaScript, this project utilizes a weather API to fetch accurate weather data, including temperature, humidity, wind speed, and weather conditions. Users can easily search for their desired city and receive up-to-date information, enhancing their ability to plan their day effectively.

Movie Search App

The Movie Search App is a web application that allows users to explore and discover movies easily. Built using HTML, CSS, and JavaScript, this project features a user-friendly interface where users can search for their favorite films, view detailed information, and access movie ratings. By integrating an external movie database API, the app provides up-to-date information on movie releases, genres, and reviews, enhancing the overall movie-watching experience.

Food Explorer

The Food Explorer Project is designed to make discovering and exploring food items simple and efficient. Built with Lightning Web Components (LWC), it integrates external APIs to dynamically fetch and display food details, offering users an engaging way to explore various cuisines. By avoiding the creation of additional Salesforce objects, the project remains lightweight and performance-focused, delivering a seamless and flexible user experience.
